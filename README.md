# 🛒 E-Commerce Project

[![Status](https://img.shields.io/badge/Status-Completed-success?style=for-the-badge)](https://github.com/AnymChac/ebac-ecommerce)

## 📝 Descripción
Este proyecto es una plataforma de tienda en línea diseñada para simular la experiencia completa de un usuario en un entorno de comercio electrónico. Se centra en una interfaz intuitiva y una navegación fluida, permitiendo gestionar un catálogo de productos.

Desarrollado como parte de mi formación técnica, este repositorio demuestra habilidades en la estructuración de aplicaciones web comerciales y manejo de lógica de negocio en el frontend.

## 🛠️ Tecnologías Utilizadas
- **Lenguajes:** HTML5, CSS3, JavaScript (ES6+).
- **Diseño:** Responsive Design para adaptabilidad en móviles y escritorio.
- **Control de Versiones:** Git & GitHub.

## 🚀 Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/AnymChac/ebac-ecommerce.git](https://github.com/AnymChac/ebac-ecommerce.git)
2. **Ejecución local:**
   No requiere servidores complejos. Simplemente abre index.html en tu navegador o utiliza Live Server en VS Code para visualizar los cambios en tiempo real.

## 📖 Ejemplos de Uso
**Interacción con el Catálogo**
El usuario puede explorar los productos disponibles. Cada artículo cuenta con una descripción y precio dinámico. Al interactuar con los botones de acción, el sistema responde visualmente para confirmar la selección.

**Gestión del Carrito (Add & Remove)**
Una de las funcionalidades core es el carrito de compras:
Agregar: Al hacer clic en "Añadir al carrito", el contador global se actualiza y el producto se almacena temporalmente.
Eliminar: Dentro del resumen de compra, el usuario puede quitar productos específicos, lo que recalcula automáticamente el monto total a pagar.

**Experiencia Multiplataforma**
Gracias al uso de Flexbox y Media Queries, el usuario puede realizar todo el proceso de compra tanto en una computadora de escritorio como en un smartphone sin perder la visibilidad de los elementos clave.
